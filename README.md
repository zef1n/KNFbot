# KNFbot

Telegram-бот для подбора досуга по городу, району и дате.

## Требования

- Python 3.10+
- Telegram Bot Token (получить у [@BotFather](https://t.me/BotFather))

## Установка

```bash
pip install -r requirements.txt
```

## Конфигурация

Скопируй `.env.example` в `.env` и вставь свой токен:

```bash
cp .env.example .env
```

```env
TOKEN_API=your_bot_token_here
```

## Запуск

```bash
python main.py
```

## Структура

```
├── main.py          — точка входа: бот, dispatcher, middleware, polling
├── handlers.py      — все хендлеры и FSM-состояния (Router)
├── keyboards.py     — клавиатуры, CallbackData-фабрики, маппинги городов/районов
├── middlewares.py   — AntiSpamMiddleware
├── config.py        — загрузка переменных окружения
├── .env             — секреты (не коммитить!)
├── .env.example     — шаблон конфигурации
└── requirements.txt
```

## Функции

- **Постоянное меню** внизу экрана (persistent reply keyboard)
- **Inline-клавиатуры** с кнопками для всех шагов
- **Редактирование сообщений** вместо новых (чистый UX)
- **Быстрый выбор даты** — сегодня, завтра, ближайшие выходные
- **Навигация назад** на любом шаге
- **Индикатор прогресса** — «Шаг X из 3»
- **Антиспам** на сообщениях и callback-кнопках
- **/help** и **/cancel** команды
- **Логирование** в stdout

## Сценарий использования

```
/start
  → Постоянное меню

🔍 Найти досуг
  → Шаг 1: Выбери город (Екатеринбург / ...)
    → Шаг 2: Выбери район (8 районов + Назад)
      → Шаг 3: Выбери дату (Сегодня / Завтра / Выходные / Вручную + Назад)
        → Результаты (Новый поиск / Изменить дату)
```
